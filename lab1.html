Lab 1: Importing Data

Objective

In this lab, you will learn how to import data into Power BI from different sources, focusing on Excel files and CSV files.

Steps

Part 1: Importing Excel Data

  1. Open Power BI Desktop.
  2. Click on "Get Data" in the Home tab.
  3. Select "Excel" from the list of data sources.
  4. Navigate to the sample Excel file "Sales_Data.xlsx" and select it.
  5. In the Navigator window, select the sheets or tables you want to import (e.g., "Sales" and "Products").
  6. Click "Load" to import the data into Power BI.

Part 2: Importing CSV Data

  1. Click on "Get Data" again.
  2. Select "Text/CSV" from the list of data sources.
  3. Navigate to the sample CSV file "Customer_Data.csv" and select it.
  4. In the preview window, verify that the data is being interpreted correctly.
  5. Click "Load" to import the CSV data into Power BI.

Exercise

  1. Import the provided sample Excel file "Sales_Data.xlsx" and CSV file "Customer_Data.csv".
  2. Create a simple table visualization for each imported dataset to verify the data.
  3. Ensure that date fields are recognized as dates and numeric fields as numbers.
  4. Rename at least two columns to make them more user-friendly.
