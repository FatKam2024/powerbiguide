Lab 9: Creating Mobile-Friendly Reports

Objective

In this lab, you will learn how to create and optimize reports for mobile devices in Power BI.

Steps

  1. Open your Power BI Desktop file from previous labs.
  2. Create a new report page for mobile layout:
    • Click the "+" icon to add a new page.
    • Rename it to "Mobile Overview".
  3. Switch to Mobile Layout view:
    • Go to the View tab and click "Mobile Layout".
    • Select your "Mobile Overview" page.
  4. Design for mobile:
    • Drag and resize visuals to fit the mobile screen.
    • Use a vertical layout for easy scrolling.
    • Increase font sizes for better readability on small screens.
  5. Add mobile-specific features:
    • Use buttons for navigation between different views.
    • Implement responsive visuals that adapt to screen size.
  6. Test your mobile layout:
    • Use the Power BI Mobile App or the mobile preview in Power BI Service.
    • Check both portrait and landscape orientations.

Exercise

  1. Create a mobile-optimized version of your main dashboard page.
  2. Design at least three mobile-friendly visualizations (e.g., card, bar chart, KPI).
  3. Implement a navigation system using buttons or bookmarks.
  4. Create a mobile-specific filter pane that's easy to use on touch screens.
  5. Test your mobile report on at least two different device sizes (e.g., phone and tablet).
  6. Publish your report and view it in the Power BI Mobile App (if possible).
