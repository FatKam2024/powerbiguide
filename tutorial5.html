DAX Basics in Power BI

Introduction

This tutorial covers the basics of DAX (Data Analysis Expressions) in Power BI, including creating calculated columns and measures.

Creating Calculated Columns

Learn how to create calculated columns using DAX:

  1. Open your Power BI report and go to the Data view.
  2. Click on "New Column" in the Modeling tab.
  3. Enter the DAX formula for your calculated column.
  4. Click "Enter" to create the column.

Creating Measures

Learn how to create measures using DAX:

  1. Go to the Report view.
  2. Click on "New Measure" in the Modeling tab.
  3. Enter the DAX formula for your measure.
  4. Click "Enter" to create the measure.

Common DAX Functions

Explore some common DAX functions:

  • SUM: Adds all the numbers in a column.
  • AVERAGE: Calculates the average of a column.
  • COUNT: Counts the number of rows in a table or column.
  • DISTINCTCOUNT: Counts the number of distinct values in a column.
  • IF: Performs conditional logic.
  • RELATED: Retrieves a value from a related table.
  • CALCULATE: Modifies the filter context of a calculation.
  • FILTER: Returns a table filtered down to specific rows.
  • ALL: Returns all rows in a table, ignoring any filters.

Example:

Total Sales = SUM(Sales[Amount])
Profit Margin = DIVIDE([Total Profit], [Total Sales], 0)
