Lab 8: Optimizing Report Performance

Objective

In this lab, you will learn techniques to optimize the performance of your Power BI reports.

Steps

  1. Open your Power BI Desktop file from previous labs.
  2. Use Performance Analyzer to identify slow-loading visuals:
    • Go to the View tab and click "Performance Analyzer".
    • Click "Start Recording" and interact with your report.
    • Review the results to identify performance issues.
  3. Optimize DAX measures:
    • Use variables to store and reuse complex calculations.
    • Replace CALCULATE + FILTER with CALCULATETABLE where possible.
  4. Improve data model:
    • Remove unnecessary columns and tables.
    • Use integers for relationship columns when possible.
    • Avoid bi-directional relationships unless necessary.
  5. Optimize visuals:
    • Limit the number of visuals per page.
    • Use bookmarks and buttons for drill-through instead of multiple visuals.

Exercise

  1. Use Performance Analyzer to identify the top 3 slowest loading visuals in your report.
  2. Optimize at least two DAX measures using variables and more efficient functions.
  3. Review your data model and make at least three improvements for performance.
  4. Redesign one report page to use fewer visuals with bookmarks for different views.
  5. Implement incremental refresh for your largest table (if applicable).
  6. Re-run Performance Analyzer and document the improvements in load times.
