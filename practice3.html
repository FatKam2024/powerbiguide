DAX and Calculations Questions

Test your knowledge of DAX and calculations with these questions:

  1. What is the difference between the SUM and SUMX functions in DAX?
  2. How do you create a calculated column in Power BI?
  3. Explain how the FILTER function works in DAX.
  4. What is the purpose of using variables in DAX?
  5. How do you use the DATEADD function in DAX for time intelligence?
  6. What is the difference between COUNTROWS and DISTINCTCOUNT in DAX?
  7. How do you create a running total using DAX?
  8. Explain the concept of context transition in DAX.
  9. How do you use the ALL function in DAX?
  10. What is the purpose of the RANKX function in DAX?

Answers

  1. SUM adds all the numbers in a column, while SUMX evaluates an expression for each row in a table and then sums the results.
  2. Go to the Data view, select the table, click "New Column" in the Modeling tab, and enter the DAX formula for the column.
  3. The FILTER function returns a table that has been filtered based on the specified criteria, allowing for complex filtering in calculations.
  4. Variables in DAX simplify complex calculations, improve performance by storing intermediate results, and make formulas more readable.
  5. Use DATEADD to shift dates by a specified number of intervals, e.g., DATEADD('Date'[Date], -1, YEAR) for the previous year.
  6. COUNTROWS counts the total number of rows in a table, while DISTINCTCOUNT counts the number of unique values in a column.
  7. Use CALCULATE with FILTER and a cumulative condition, e.g., CALCULATE(SUM(Sales[Amount]), FILTER(ALL(Date), Date[Date] <= MAX(Date[Date]))).
  8. Context transition occurs when a row context is transformed into a filter context, typically when using measures inside calculated columns.
  9. ALL is used to remove filters from a table or column, often used inside CALCULATE to modify the filter context.
  10. RANKX is used to rank items within a category based on a specified expression, useful for creating top N or bottom N reports.
