Advanced DAX in Power BI

Introduction

This tutorial covers advanced DAX topics in Power BI, including time intelligence functions, complex calculations, and iterators.

Time Intelligence Functions

Learn how to use DAX time intelligence functions:

  • DATEADD: Shifts dates by a specified number of intervals.
  • PARALLELPERIOD: Returns a parallel period for a given date.
  • SAMEPERIODLASTYEAR: Returns the same period in the previous year.
  • YTD: Year-to-date calculations.
  • MTD: Month-to-date calculations.
  • QTD: Quarter-to-date calculations.

Example:

Sales YTD = CALCULATE([Total Sales], DATESYTD(DimDate[Date]))

Complex Calculations

Learn how to perform complex calculations using DAX:

  1. Use the CALCULATE function to modify filter context.
  2. Combine multiple DAX functions to create advanced measures.
  3. Use variables in DAX to simplify complex formulas.

Example:

Market Share = 
VAR TotalSales = CALCULATE([Total Sales], ALL(Product))
RETURN
DIVIDE([Total Sales], TotalSales)

Iterators and X-Functions

Learn how to use iterators and X-functions for row-by-row calculations:

  • SUMX: Calculates the sum of an expression evaluated for each row.
  • AVERAGEX: Calculates the average of an expression evaluated for each row.
  • MAXX and MINX: Find the maximum or minimum value of an expression.
  • FILTER: Returns a table filtered down to specific rows.
  • ADDCOLUMNS: Adds calculated columns to a table expression.

Example:

Total Profit = SUMX(Sales, Sales[Quantity] * (Sales[Price] - RELATED(Product[Cost])))
