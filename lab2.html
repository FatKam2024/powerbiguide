Lab 2: Data Modeling

Objective

In this lab, you will learn how to create relationships between tables and build a data model in Power BI, implementing a star schema design.

Steps

  1. Open Power BI Desktop and load the data from Lab 1 (Sales, Products, and Customers).
  2. Go to the Model view by clicking on the Model icon in the left sidebar.
  3. Create relationships:
    • Drag the "ProductID" field from the Sales table to the "ProductID" field in the Products table.
    • Drag the "CustomerID" field from the Sales table to the "CustomerID" field in the Customers table.
  4. Configure relationship properties:
    • Double-click on each relationship line.
    • Ensure the cardinality is set to "Many to One" (*:1).
    • Set the Cross filter direction to "Single".
  5. Create a Date table:
    • Go to the "Modeling" tab and click "New Table".
    • Enter the DAX formula: Date = CALENDAR(DATE(2020,1,1), DATE(2023,12,31))
  6. Create a relationship between the Sales table and the new Date table.

Exercise

  1. Build the data model as described in the steps above.
  2. Create two calculated columns:
    • In the Products table, create a "Profit" column (Sell Price - Cost Price).
    • In the Date table, create a "Month Name" column using the FORMAT function.
  3. Verify all relationships are correctly set up in the model view.
  4. Hide unnecessary columns from report view (e.g., IDs used for relationships).
