Lab 3: Creating Visualizations

Objective

In this lab, you will learn how to create various visualizations in Power BI using the data model you built in Lab 2.

Steps

  1. Open your Power BI report from Lab 2.
  2. Create a bar chart:
    • Drag "Product Name" to the Axis field.
    • Drag "Sales Amount" to the Values field.
    • Sort the chart by Sales Amount in descending order.
  3. Create a line chart:
    • Drag "Date" to the Axis field.
    • Drag "Sales Amount" to the Values field.
    • Add "Product Category" to the Legend field for multiple lines.
  4. Create a map visualization:
    • Drag "Customer City" to the Location field.
    • Drag "Sales Amount" to the Size field.
  5. Create a pie chart:
    • Drag "Product Category" to the Legend field.
    • Drag "Profit" to the Values field.
  6. Customize each visualization using the Format pane:
    • Change colors, fonts, and titles.
    • Add data labels where appropriate.
    • Adjust axes and legends for clarity.

Exercise

  1. Create the four visualizations described in the steps above.
  2. Add a slicer for filtering by date.
  3. Create a card visual showing total sales.
  4. Arrange the visualizations on the report page in a logical and visually appealing manner.
  5. Ensure all visualizations interact with each other (cross-filtering).
  6. Add a title to your report page.
