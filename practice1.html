Practice Questions: Set 1 (Basics)

Basic Questions

Test your understanding of Power BI basics with these questions:

  1. What are the three main views in Power BI Desktop?
  2. How do you import data from an Excel file into Power BI?
  3. What is the purpose of the Filters pane in Power BI?
  4. Explain the difference between a report and a dashboard in Power BI.
  5. What types of visualizations can you create in Power BI?
  6. How do you refresh data in Power BI Desktop?
  7. What is the difference between Power BI Desktop and Power BI Service?
  8. How do you create a calculated column in Power BI?
  9. What is the purpose of the Query Editor in Power BI?
  10. How do you share a Power BI report with others?

Answers

  1. Report View, Data View, Model View
  2. Click on "Get Data", select "Excel", choose the file, select the tables/sheets, click "Load".
  3. The Filters pane allows you to filter data in your visualizations at various levels (visual, page, report).
  4. A report is a collection of visualizations on one or more pages, while a dashboard is a single-page canvas, often with live data and tiles from multiple reports.
  5. Bar charts, line charts, pie charts, maps, tables, scatter plots, waterfall charts, KPIs, and many more.
  6. Click on "Refresh" in the Home tab, or right-click on a table in the Fields pane and select "Refresh data".
  7. Power BI Desktop is a Windows application for creating reports and datasets, while Power BI Service is a cloud-based platform for sharing and viewing reports.
  8. In the Data view, select the table, click "New Column" in the Modeling tab, and enter a DAX formula.
  9. The Query Editor is used for data transformation and cleaning before loading data into Power BI.
  10. Publish the report to Power BI Service, then use the "Share" button to invite others or create a shareable link.
