Power BI Data Transformation

Introduction

This tutorial covers data transformation techniques in Power BI, including cleaning, shaping, and combining data using Power Query.

Power Query Editor

Power Query is the data transformation engine in Power BI:

  1. Open Power BI Desktop and load your data.
  2. Click on "Transform Data" to open the Power Query Editor.
  3. Use the Query Settings pane to view and modify applied steps.

Cleaning Data

Learn how to clean your data using the Power Query Editor:

  1. Use tools like "Remove Rows," "Replace Values," and "Split Column" to clean your data.
  2. Handle errors and empty values with the "Remove Errors" and "Fill" options.

Shaping Data

Learn how to shape your data to meet your analysis needs:

  1. Use the "Pivot Column" and "Unpivot Columns" features to reshape your data.
  2. Group data using the "Group By" function.
  3. Create custom columns using the "Add Column" feature and M language.

Combining Data

Learn how to combine data from multiple sources:

  1. Use "Append Queries" to combine tables vertically.
  2. Use "Merge Queries" to combine tables horizontally based on a key column.
  3. Use "Reference" to create a new query based on an existing one.
